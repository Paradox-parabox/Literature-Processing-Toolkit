# 数学视角PICOS框架

## 概述

PICOS框架源自医学领域，但数学研究的知识体系与研究范式与医学有本质区别。本文档提供数学领域（包括纯数学、应用数学、计算数学、统计学、数学交叉学科）的PICOS适配标准。

## 数学PICOS框架定义

| 要素 | 名称 | 定义 | 必要性 |
|------|------|------|--------|
| **P** | Problem（问题域） | 研究的数学问题及其背景 | 必需 |
| **I** | Instrument（方法/工具） | 解决问题的具体数学方法或算法 | 必需 |
| **C** | Comparison（对比参照） | 与现有方法/结果的对比 | 可选 |
| **O** | Outcomes（产出） | 研究得到的数学结论和应用价值 | 必需 |
| **S** | Study design（研究范式） | 研究的整体方法论框架 | 必需 |

## 各要素详细定义

### P - Problem（问题域）

**定义**：研究的核心数学问题，可包含应用背景。

**分类**：

| 类型 | 说明 | 示例 |
|------|------|------|
| **纯数学问题** | 抽象数学结构的研究 | 紧黎曼流形上的Laplace算子谱理论 |
| **方程问题** | 微分方程/积分方程的研究 | 分数阶Cahn-Hilliard方程的适定性 |
| **算法问题** | 数值方法的研究 | 高维积分的快速算法 |
| **建模问题** | 数学建模与应用 | 多孔介质多相流的数学模型 |
| **统计问题** | 统计推断问题 | 高维数据的变量选择 |

**提取要点**：
1. 明确问题的数学性质（方程类型、结构类型等）
2. 识别问题的应用背景（如有）
3. 确定问题的数学分类（分析、代数、几何、概率等）

---

### I - Instrument（方法/工具）

**定义**：解决问题的具体数学方法、算法或技术手段。

**⚠️ 关键区分：核心方法 vs 对比参照**

在提取I要素时，必须严格区分**核心方法**和**对比参照**，避免"方法稀释"导致检索灾难。

#### 判断流程

```
识别到方法M
    ↓
检查正文对M的态度
    ├── 明确采用/发展/改进 → I-核心方法
    ├── 指出局限性/作为对标 → C-对比参照（非I要素！）
    ├── 提及但未明确表态 → I-相关方法（低优先级）
    └── 作为数据来源 → S-特殊类型（如实验数据）
```

#### 判断依据表

| 正文态度 | 方法角色 | PICOS要素 | 示例 |
|---------|---------|----------|------|
| "采用""发展""构建""提出" | **核心方法** | I | "构建LB模型" → LBM是核心方法 |
| "但是...困难""仅适用于""局限" | **对比参照** | C | "有限元矩阵装配复杂" → 有限元是靶子 |
| "相比...""优于""区别于" | **对比参照** | C | "比传统方法更快" → 传统方法是靶子 |
| "也可用""另一种方法" | **相关方法** | I（低优先级） | "也可用有限差分" → 低优先级 |
| "实验数据""测量结果" | **数据来源** | S | "通过实验数据反演" → 实验是数据源 |

#### 典型错误示例

**错误1：方法稀释**
```
原文："有限元方法矩阵装配复杂，推广到高维困难，而LBM方法边界处理简单。"

❌ 错误处理：
I: 有限元方法、LBM方法  ← 合并处理，导致检索时被海量有限元文献淹没

✅ 正确处理：
C: 有限元方法（对比参照）→ 局限：矩阵装配复杂，推广高维困难
I: LBM方法（核心方法）→ 优势：边界处理简单
```

**错误2：遗漏数据源**
```
原文："通过实验数据建立反演分数阶数的PINNs模型。"

❌ 错误处理：
排除：纯实验研究  ← 会误杀PINNs需要的数据源

✅ 正确处理：
S: 接受提供高质量实验数据的研究（用于PINNs训练验证）
I: PINNs（核心方法）
```

#### 分类

| 类型 | 说明 | 示例 |
|------|------|------|
| **理论方法** | 数学分析技术 | 变分法、渐近分析、不动点定理 |
| **算法方法** | 数值计算方法 | LBM、相场方法、PINNs |
| **构造方法** | 构造性技术 | 显式构造、递归构造 |
| **证明技术** | 证明方法 | 归纳法、反证法、构造性证明 |
| **统计方法** | 统计技术 | 极大似然估计、贝叶斯推断 |

**提取要点**：
1. 区分核心方法和辅助方法
2. 识别方法的创新点
3. 明确方法的适用范围

---

### C - Comparison（对比参照）

**定义**：与现有方法、理论或结果的对比分析。

**重要性**：**此要素在数学研究中为可选**，原因如下：
- 纯数学研究可能是开创性的，无现有对比
- 某些理论构建是独立的，不涉及方法对比

**分类**：

| 类型 | 说明 | 示例 |
|------|------|------|
| **方法对比** | 与现有算法对比 | 比传统有限元方法计算效率提高X倍 |
| **理论对比** | 与已有结果对比 | 改进了XXX的误差估计 |
| **精度对比** | 与精确解对比 | 与解析解的误差为O(h²) |
| **验证对比** | 与实验数据对比 | 模拟结果与实验数据吻合 |

**提取要点**：
1. 明确对比的对象
2. 说明对比的维度（效率、精度、适用范围等）
3. 量化对比结果（如有）

---

### O - Outcomes（产出）

**定义**：研究得到的数学结论和应用价值。

**双层产出结构**：

| 层次 | 类型 | 说明 | 示例 |
|------|------|------|------|
| **数学产出** | 存在性 | 解的存在性/唯一性 | 证明了弱解的存在唯一性 |
| **数学产出** | 性质 | 数学性质 | 收敛性、稳定性、能量守恒 |
| **数学产出** | 估计 | 定量估计 | 收敛阶为O(h²) |
| **数学产出** | 分类 | 结构分类 | 完成了XXX的分类 |
| **应用产出** | 模拟 | 数值模拟结果 | 得到了渗透率分布 |
| **应用产出** | 验证 | 方法验证 | 与实验数据吻合良好 |
| **应用产出** | 应用 | 实际应用 | 应用于油藏开发优化 |

**提取要点**：
1. 区分数学产出和应用产出
2. 明确产出的可验证性
3. 量化产出指标（如有）

---

### S - Study design（研究范式）

**定义**：研究的整体方法论框架，描述研究是如何开展的。

**分类**：

| 类型 | 说明 | 典型结构 |
|------|------|---------|
| **纯理论研究** | 纯数学证明 | 定义→定理→证明→推论 |
| **算法设计研究** | 新算法开发 | 问题→算法→分析→实验 |
| **理论分析研究** | 方程/模型分析 | 模型→分析→性质→结论 |
| **数值实验研究** | 计算验证 | 方法→实现→测试→分析 |
| **应用验证研究** | 应用检验 | 方法→应用→验证→结论 |
| **混合研究** | 多范式组合 | 理论+算法+实验 |

**提取要点**：
1. 识别主要研究范式
2. 明确各范式的权重
3. 判断研究的完整性

---

## 数学各子领域的PICOS适配

### 纯数学

| 要素 | 适配要点 |
|------|---------|
| **P** | 数学结构（群、环、域、拓扑、流形等） |
| **I** | 证明方法、构造技术 |
| **C** | 可选，与已有定理/结果对比 |
| **O** | 定理、引理、推论、反例、分类 |
| **S** | 纯理论研究 |

**示例**：
- P: 紧黎曼流形上的Laplace算子
- I: 谱分析方法、热核技术
- C: 改进了Weyl渐近公式
- O: 谱的渐近性质、特征值的渐近估计
- S: 纯理论研究

---

### 计算数学/数值分析

| 要素 | 适配要点 |
|------|---------|
| **P** | 数值问题（方程离散化、积分计算等） |
| **I** | 数值算法（LBM、有限元、有限差分等） |
| **C** | 与现有算法对比（效率、精度） |
| **O** | 收敛性、稳定性、精度、复杂度 |
| **S** | 算法设计+理论分析+数值实验 |

**示例**：
- P: 分数阶偏微分方程的数值求解
- I: 格子Boltzmann方法
- C: 比有限差分方法精度更高
- O: 二阶收敛、无条件稳定、能量守恒
- S: 算法设计+理论分析+数值实验

---

### 应用数学

| 要素 | 适配要点 |
|------|---------|
| **P** | 数学问题+应用背景 |
| **I** | 分析方法、建模技术 |
| **C** | 与已有模型/方法对比 |
| **O** | 数学性质+应用解释 |
| **S** | 理论分析+应用验证 |

**示例**：
- P: 多孔介质多相流的数学建模
- I: 相场方法、渐近分析
- C: 比传统达西模型更准确
- O: 界面动力学、渗透率预测
- S: 理论分析+数值验证

---

### 统计学

| 要素 | 适配要点 |
|------|---------|
| **P** | 统计推断问题 |
| **I** | 统计方法、估计技术 |
| **C** | 与现有方法对比 |
| **O** | 估计性质、渐近行为 |
| **S** | 理论推导+模拟研究/应用分析 |

**示例**：
- P: 高维数据的变量选择
- I: 惩罚似然方法
- C: 比LASSO更稳定
- O: 选择一致性、Oracle性质
- S: 理论推导+模拟研究

---

### 数学交叉学科

| 要素 | 适配要点 |
|------|---------|
| **P** | 数学问题+应用领域问题 |
| **I** | 数学方法+领域方法 |
| **C** | 多维度对比 |
| **O** | 数学贡献+应用价值 |
| **S** | 混合研究范式 |

**示例**：
- P: 非均匀多孔介质多相流输运
- I: LBM+非局部相场模型+PINNs
- C: 与传统局部模型对比
- O: 收敛性（数学）+渗透率预测（应用）
- S: 算法设计+理论分析+数值实验+应用验证

## 纳入/排除标准生成指南

### 纳入标准

基于PICOS要素生成：

```markdown
## 纳入标准

### P (问题域)
- [ ] 数学问题符合：{具体问题类型}
- [ ] 应用背景符合：{具体应用领域}（如适用）

### I (方法/工具)
- [ ] 采用核心方法：{具体方法}
- [ ] 或采用相关方法：{相关方法列表}

### O (产出)
- [ ] 数学产出涉及：{具体数学性质}
- [ ] 应用产出涉及：{具体应用指标}（如适用）

### S (研究范式)
- [ ] 研究类型符合：{理论研究/算法设计/数值实验等}
```

### 排除标准

基于PICOS要素的反面：

```markdown
## 排除标准

### 问题域不符
- 不涉及{具体数学问题}的研究
- 应用背景不符的研究

### 方法不符
- 不采用{核心方法}的研究
- 采用{排除方法}的研究

### 研究范式不符
- 纯综述文章
- 无数学贡献的应用研究
- 无数值验证的算法研究（视要求而定）
```

## 与医学PICOS的关键区别

| 方面 | 医学PICOS | 数学PICOS |
|------|----------|-----------|
| **P要素** | 患者群体 | 数学问题+可选应用背景 |
| **I要素** | 干预措施 | 方法/工具/算法 |
| **C要素** | 必需（对照组） | 可选（纯数学可能无对照） |
| **O要素** | 临床结局 | 数学产出+可选应用产出 |
| **S要素** | RCT/队列等 | 理论/算法/实验等范式 |

---

## 使用建议

1. **识别研究类型**：首先判断是纯数学、计算数学还是交叉学科
2. **灵活应用C要素**：纯数学研究C要素可能为空
3. **区分产出层次**：明确数学产出和应用产出
4. **判断研究范式**：S要素描述整体方法论框架
